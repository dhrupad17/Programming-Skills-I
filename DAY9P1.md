# To Lower Case
---
- ## Question:
> Given a string s, return the string after replacing every uppercase letter with the same lowercase letter.
---
- ## Example:
> Input: s = "Hello"
> 
> Output: "hello"
---
- ## Solution:
**Code :**
```java
class Solution {
    public String toLowerCase(String s) {
        // StringBuilder str=new StringBuilder(s);
        return s.toLowerCase();
        
    }
}
